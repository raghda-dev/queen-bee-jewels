# QueenBee Systems

This is a full-stack ecommerce app built with:
- **Client**: Next.js (frontend)
- **Server**: Express.js (backend)
- **Storefront**: Shopify API
# 🐝 Queen Bee Jewels 💎

**Queen Bee Jewels** is a luxurious full-stack e-commerce platform crafted to deliver an elegant and seamless online shopping experience for jewelry lovers.  
It combines a custom backend (for users, authentication, carts, wishlists) with Shopify’s product/collection API for scalability, while ensuring a polished UI with refined animations and responsive layouts.  

---

## ✨ Features
- **Luxury UI & Animations** – crafted with Tailwind CSS + SCSS for elegance and detail.  
- **Authentication System** – secure signup/login, JWT sessions, and protected routes.  
- **User Accounts** – manage profiles, shopping carts, and wishlists.  
- **Shopify Integration** – product & collection data fetched dynamically from Shopify.  
- **Responsive Design** – optimized for desktop, tablet, and mobile.  
- **State Management** – smooth and scalable with Redux Toolkit.  

---

## 🛠️ Tech Stack

### **Frontend**
- **Next.js (App Router)** – modern React framework with SSR & SEO benefits.  
- **TypeScript** – strong typing for scalability.  
- **Tailwind CSS** – utility-first styling.  
- **SCSS** – structured styling for themes and variables.  
- **Framer Motion** – elegant animations.  

### **Backend**
- **Node.js + Express.js** – RESTful backend API.  
- **MongoDB + Mongoose** – database for users, carts, and wishlists.  
- **JWT Authentication** – secure login & authorization.  

### **Integration**
- **Shopify Storefront API** – fetch products & collections.
